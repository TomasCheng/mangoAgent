# MangoAgent 项目总结

## 项目简介
MangoAgent 是一个基于 Python 的自主 AI 代理系统，专为多任务代码开发和团队协作设计。项目采用模块化架构，支持 Git 工作树管理、任务分配、技能加载和事件驱动通信。

## 核心特性

### 🚀 多任务并行处理
- 支持创建和管理多个独立任务
- 每个任务可绑定到独立的 Git 工作树
- 任务状态跟踪（待处理、进行中、已完成）

### 🌳 Git 工作树管理
- 动态创建和管理 Git 工作树
- 工作树生命周期管理
- 工作树状态监控和事件跟踪

### 👥 团队协作
- 支持创建和管理团队成员
- 消息传递和广播机制
- 任务分配和协调

### 📚 技能系统
- 模块化技能知识库
- 运行时技能动态加载
- 内置技能：代理设计、提示工程、Git 工作流、代码审查、PDF 生成

### 🔌 事件驱动架构
- 发布-订阅模式的事件系统
- 工作树和任务生命周期事件
- 模块间解耦通信

## 技术架构

### 核心模块
```
src/mangoAgent/
├── core/                    # 核心引擎
│   ├── agent.py            # 主代理类
│   ├── worktree_manager.py # 工作树管理
│   ├── task_manager.py     # 任务管理
│   └── event_bus.py        # 事件总线
├── tools/                  # 工具系统
│   ├── base.py            # 工具基类
│   └── ...                # 20+ 内置工具
├── skills/                # 内置技能
│   ├── git_workflow/      # Git 工作流
│   ├── code_review/       # 代码审查
│   └── pdf_generation/    # PDF 生成
└── __main__.py            # 程序入口
```

### 数据存储
```
work-space/
├── .worktrees/           # 工作树存储
├── .tasks/              # 任务存储
├── .team/               # 团队配置
└── skills/              # 工作空间技能
```

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 配置环境
```bash
cp .env.example .env
# 编辑 .env 文件，添加 API 密钥等配置
```

### 启动代理
```bash
python run.py
```

### 基本使用示例
```python
# 创建任务
task_create("开发新功能", "实现用户认证模块")

# 创建工作树
worktree_create("auth-feature", task_id=1)

# 在工作树中执行命令
worktree_run("auth-feature", "git status")

# 加载技能
load_skill("git_workflow")

# 创建团队成员
spawn_teammate("backend-dev", "后端开发", "负责API开发")
```

## 主要工具函数

### 任务管理
- `task_create()` - 创建新任务
- `task_list()` - 列出所有任务
- `task_update()` - 更新任务状态
- `task_bind_worktree()` - 绑定任务到工作树

### 工作树管理
- `worktree_create()` - 创建工作树
- `worktree_list()` - 列出工作树
- `worktree_run()` - 在工作树中执行命令
- `worktree_remove()` - 删除工作树

### 团队协作
- `spawn_teammate()` - 创建团队成员
- `send_message()` - 发送消息
- `broadcast()` - 广播消息
- `read_inbox()` - 读取收件箱

### 技能管理
- `load_skill()` - 加载技能
- `list_skills()` - 列出可用技能
