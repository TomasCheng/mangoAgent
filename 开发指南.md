## 项目结构说明

### 核心模块开发

#### Agent 类 (`src/mangoAgent/core/agent.py`)
- 主代理类，协调所有功能
- 负责工具注册、技能加载、事件处理
- 主要方法：
  - `__init__()`: 初始化所有管理器
  - `register_tool()`: 注册工具
  - `load_skill()`: 加载技能
  - `execute()`: 执行命令

#### 工具开发指南

**创建新工具步骤：**

1. 在 `src/mangoAgent/tools/` 下创建新文件，如 `my_tool.py`

2. 实现工具类：
```python
from .base import Tool

class MyTool(Tool):
    name = "my_tool"
    description = "我的自定义工具描述"
    
    def execute(self, **kwargs):
        # 工具逻辑实现
        param1 = kwargs.get("param1")
        # ... 执行操作
        return {"result": "success", "data": ...}
```

3. 在 `src/mangoAgent/tools/__init__.py` 中注册：
```python
from .my_tool import MyTool

__all__ = [
    # ... 其他工具
    "MyTool",
]
```

4. 工具会自动在 Agent 初始化时加载

#### 技能开发指南

**创建新技能步骤：**

1. 在 `.mango/skills/` 下创建技能目录，如 `my_skill/`

2. 创建 `SKILL.md` 文件：
```markdown
# 我的技能

## 概述
技能描述...

## 使用场景
- 场景1
- 场景2

## 最佳实践
1. 实践1
2. 实践2

## 示例
```python
# 示例代码
```

## 注意事项
- 注意1
- 注意2
```

3. 技能会在调用 `load_skill("my_skill")` 时加载

class LazyLoader:
    def __init__(self):
        self._loaded = False
        self._data = None
    
    @property
    def data(self):
        if not self._loaded:
            self._load_data()
        return self._data

### 2. 错误恢复
```python
def safe_execute_tool(tool_name, **kwargs):
    """安全执行工具"""
    try:
        tool = self.tools[tool_name]
        return tool.execute(**kwargs)
    except KeyError:
        return {"error": f"工具 {tool_name} 不存在"}
    except Exception as e:
        return {"error": f"执行失败: {str(e)}"}
```

## 常见问题

### Q1: 如何添加新的 API 集成？
A: 创建新的工具类，在 `execute` 方法中实现 API 调用逻辑。

### Q2: 技能文件应该放在哪里？
A: 项目级技能放在 `.mango/skills/`，工作空间特定技能放在 `work-space/skills/`。

### Q3: 如何调试工具执行？
A: 使用事件总线监听工具执行事件，或添加调试日志。

### Q4: 支持哪些 Python 版本？
A: 目前支持 Python 3.8+，建议使用 Python 3.9+。
