# MangoAgent 项目总结文档

## 项目概述

MangoAgent 是一个基于 Python 的自主 AI 代理系统，设计用于多任务协作和代码开发工作流管理。项目采用模块化架构，支持工作树管理、任务分配、技能加载和团队协作等功能。

## 项目架构

### 核心模块

1. **Agent 核心 (`src/mangoAgent/core/agent.py`)**
   - 主代理类，负责协调所有功能
   - 支持工具调用、技能加载、任务管理
   - 提供事件驱动架构

2. **工作树管理器 (`src/mangoAgent/core/worktree_manager.py`)**
   - 管理 Git 工作树的创建、删除和状态跟踪
   - 支持任务与工作树的绑定
   - 提供工作树生命周期管理

3. **任务管理器 (`src/mangoAgent/core/task_manager.py`)**
   - 任务创建、更新和状态跟踪
   - 支持多任务并行处理
   - 任务与工作树的关联管理

4. **事件总线 (`src/mangoAgent/core/event_bus.py`)**
   - 发布-订阅模式的事件系统
   - 支持模块间解耦通信
   - 工作树和任务生命周期事件跟踪

### 工具系统 (`src/mangoAgent/tools/`)
- **基础工具类 (`base.py`)**: 定义工具接口和注册机制
- **内置工具**: bash、文件操作、任务管理、工作树操作等
- **扩展性**: 支持自定义工具开发

### 技能系统 (`src/mangoAgent/skills/` 和 `./.mango/skills/`)
- **技能目录结构**: 每个技能包含 SKILL.md 描述文件
- **内置技能**:
  - `agent_design`: 自主代理设计模式
  - `prompt_engineer`: 提示工程技巧
  - `git_workflow`: Git 工作流
  - `code_review`: 代码审查清单
  - `pdf_generation`: PDF 生成技能
- **技能加载**: 运行时动态加载技能知识

### 配置系统 (`src/mangoAgent/config.py`)
- 环境变量管理
- API 密钥配置
- 项目路径配置

## 主要功能特性

### 1. 多任务并行处理
- 支持创建多个独立任务
- 每个任务可绑定到独立的工作树
- 任务状态跟踪（pending、in_progress、completed）

### 2. Git 工作树管理
- 动态创建和管理 Git 工作树
- 工作树生命周期管理（创建、使用、保留、删除）
- 工作树状态监控

### 3. 团队协作
- 支持创建和管理团队成员
- 消息传递和广播机制
- 任务分配和协调

### 4. 技能知识库
- 模块化技能系统
- 技能文档化存储
- 运行时技能加载和应用

### 5. 事件驱动架构
- 工作树创建/删除事件
- 任务状态变更事件
- 系统操作日志记录

## 技术栈

### 编程语言
- **Python 3.x**: 主要开发语言

### 核心依赖
- **anthropic**: Claude API 客户端
- **python-dotenv**: 环境变量管理
- **rich**: 终端美化输出
- **prompt_toolkit**: 交互式命令行界面

### 开发工具
- **Git**: 版本控制和多工作树管理
- **JSON**: 配置和数据存储格式
- **Markdown**: 文档和技能描述

## 使用方式

### 启动代理
```bash
python run.py
```

### 主要操作命令
1. **任务管理**:
   - `task_create`: 创建新任务
   - `task_list`: 列出所有任务
   - `task_update`: 更新任务状态

2. **工作树管理**:
   - `worktree_create`: 创建工作树
   - `worktree_list`: 列出工作树
   - `worktree_run`: 在工作树中执行命令

3. **团队协作**:
   - `spawn_teammate`: 创建团队成员
   - `send_message`: 发送消息
   - `broadcast`: 广播消息

4. **技能管理**:
   - `load_skill`: 加载技能
   - `list_skills`: 列出可用技能
